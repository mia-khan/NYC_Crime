1. Crime Trends Over Each Quarter of NYC Parks in 2023 & 2024

How to Use the Interactive Crime Trends Visualization

This interactive line chart displays crime trends over time in New York City parks. It is embedded directly into the webpage using an Altair-generated HTML chart.

Filter by Crime Type

  • Below the chart, you'll find a row of white radio buttons labeled with different crime categories.
  • By clicking any crime type, the chart will update instantly to show only that type over time.
  • The ‚ÄúAll‚Äù option resets the view to display trends for all crime types together.

What Crime Types Are Available?

The chart allows you to explore trends for:

  • Murder
  • Rape
  • Robbery
  • Felony Assault
  • Burglary
  • Grand Larceny
  • Grand Larceny of Motor Vehicle

Each one corresponds to a distinct color-coded line in the chart (when "All" is selected).

What the Chart Shows

  • The x-axis represents time, broken down by quarters from Q1 2023 to Q4 2024.
  • The y-axis shows the total number of crimes reported for each selected type during each quarter.
  • All data is aggregated citywide and sourced from official NYC crime statistics reported in public parks.

This visualization helps highlight:

  • Which crimes are increasing or decreasing over time
  • Seasonal or cyclical crime patterns (e.g., spikes in summer)
  • Differences in frequency across crime types (e.g., grand larceny is consistently higher)

Explore Further

You can:

  • Toggle different crime types to compare them
  • Observe specific quarters with unusual spikes or drops
  • Use this chart alongside borough-level and stacked visualizations elsewhere on the site
